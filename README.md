# MYHamGuide
This is a guide page for how to get MCMC Radio Amateur License in Malaysia.  
  
>[!IMPORTANT]
>In order to obtain a license, you must first sit for the RAE (Radio Amateur Examination).

>[!NOTE]
>In Malaysia, there are three license classes for amateur radio.  
>- Class A -> Highest class among the three, has full privilege to operate all ham radio bands assigned to Malaysia, with the highest power limit.  
>- Class B -> Intermediate class among the three, has privilege to operate most HF frequencies + full VHF/UHF, with almost all gazetted bands in 50W power limit.  
>- Class C -> Entry-level class among the three, has limited privilege to only VHF/UHF, with 25W power limit.
